# Kernel "Zenplus" - A Nix OS Kernel Flake with certain optimisations for NixOS and Zen 4
# NOTE - As of 18/05/26 This repo builds for the Zen 4 Architecture ONLY 
[![Update and Build Kernel](https://github.com/tj5miniop/linux-zenplus-nix/actions/workflows/build.yml/badge.svg)](https://github.com/tj5miniop/linux-zenplus-nix/actions/workflows/build.yml)
#### A personal flake for NixOS which supplies the Zen kernel from nixos-unstable along with patches from the Linux Gaming Community; automatically pushes to Cachix
# Credits

## How to use 
#### A custom linux kernel for NixOS patched with gaming related utilities such as 

- BORE Scheduler (Used by CachyOS) (SCX-LAVD enabled in my configuration)
- OpenRGB patches 
- Gamescope patches
- Zen 4 Based Optimisations
